# 🍜 Thai Restaurant Landing Page

เว็บไซต์หน้าเดียว (Single Page Website) สำหรับร้านอาหารไทย ตกแต่งสไตล์มินิมัล สะอาดตา มาพร้อมกับเมนูอาหารยอดนิยมและระบบฟอร์มสั่งซื้อที่ใช้งานง่าย

---

## ✨ คุณสมบัติของเว็บไซต์ (Features)

* **Header & Navigation Bar:** ระบบแถบเมนูบาร์ด้านบนที่สามารถคลิกเพื่อเลื่อน (Anchor Link) ไปยังส่วนต่าง ๆ ของหน้าเว็บได้อย่างรวดเร็ว
* **Menu Section:** แสดงรายการอาหารพร้อมรูปภาพ ชื่อเมนู ราคา และปุ่มทางลัด "Order Now" เพื่อไปยังฟอร์มสั่งซื้อทันที
* **Order Form:** ฟอร์มรับข้อมูลการสั่งซื้อที่รองรับการระบุชื่อ, เมนูอาหาร, จำนวน (จำกัด 0-10 จาน), ที่อยู่จัดส่ง และกล่องยืนยันการชำระเงิน
* **Responsive Design:** ตัวฟอร์มและองค์ประกอบต่าง ๆ จัดวางอย่างเป็นระเบียบ รองรับการกรอกข้อมูลได้เป็นอย่างดี

---

## 🛠️ เทคโนโลยีที่ใช้ (Technologies Used)

โปรเจกต์นี้พัฒนาขึ้นโดยใช้เทคโนโลยีเว็บพื้นฐานมาตรฐาน ดังนี้:

| เทคโนโลยี | หน้าที่และบทบาทในโปรเจกต์ |
| :--- | :--- |
| **HTML5** | โครงสร้างหลักของหน้าเว็บ, ลิงก์ภายในหน้า (Anchors) และองค์ประกอบฟอร์ม (`<form>`) |
| **CSS3** | การปรับแต่งสไตล์ (Internal Style), ตกแต่งสีสันของ Navigation Bar และเอฟเฟกต์ `:hover` เปลี่ยนสีเมื่อชี้เมาส์ |

---

## 📂 โครงสร้างภายในโค้ด (Code Structure)

หน้านี้ประกอบด้วยส่วนประกอบหลัก 3 ส่วนหลัก ๆ ภายใต้แท็ก `<body>`:

1.  **Navbar:** ใช้ Class `.navbar` จัดกึ่งกลางหน้าจอ ลิงก์ปลายทางเชื่อมกับไอดี `#heading-1` และ `#heading-2`
2.  **Menu (`<section id="#heading-1">`):** ส่วนแสดงเมนูอาหาร (ตัวอย่างปัจจุบันคือเมนู **Pad Thai** ราคา $12) พร้อมลิงก์ด่วนสำหรับสั่งซื้อ
3.  **Order Form (`<form id="#heading-2">`):** ฟอร์มที่ประกอบด้วย Input Types ที่หลากหลาย เช่น `text`, `number` (พร้อม Attribute `min` และ `max`), `textarea` และ `checkbox`

---

## 🚀 วิธีการนำไปใช้งาน (How to Run)

1.  คัดลอกโค้ด HTML ทั้งหมดไปไว้ในเครื่องของคุณ
2.  บันทึกไฟล์เป็นชื่อ `index.html`
3.  ดับเบิลคลิกที่ไฟล์ `index.html` เพื่อเปิดใช้งานผ่านเว็บเบราว์เซอร์ (เช่น Google Chrome, Safari, Firefox) ได้ทันที!

## 💡 ข้อแนะนำสำหรับการพัฒนาต่อ (Future Improvements)

* **แก้ไขโครงสร้าง HTML (Semantic HTML):** ปรับปรุงแท็กที่พิมพ์ผิดในโค้ดเดิม เช่น เปลี่ยน `<labe>` เป็น `<label>` และลบเครื่องหมาย `#` ออกจาก attribute `id` (เช่น เปลี่ยนจาก `id="#heading-1"` เป็น `id="heading-1"`) เพื่อให้ระบบ Anchor Link ทำงานได้อย่างถูกต้องสมบูรณ์
* **เพิ่ม JavaScript:** เชื่อมต่อระบบฟอร์มเข้ากับหลังบ้าน หรือส่งข้อมูลการสั่งซื้อผ่านไลน์ (Line Notify) หรืออีเมล
* **ปรับแต่ง CSS เพิ่มเติม:** แยกไฟล์ CSS ออกมาเป็นสไตล์ภายนอก (External Stylesheet) และทำให้เว็บไซต์รองรับ Mobile Responsive 100%
