# 🍜 Thai Restaurant Website - Codedex Final Project

ยินดีต้อนรับสู่โปรเจกต์ **Thai Restaurant Website** ซึ่งเป็นส่วนหนึ่งของโปรเจกต์จบ (Final Project) ในการเรียนรู้และฝึกฝนการเขียนภาษา HTML และ CSS บนแพลตฟอร์ม **Codedex.io** เว็บไซต์นี้จำลองหน้าต่างของร้านอาหารไทยที่มีการแสดงเมนูอาหารยอดนิยม พร้อมระบบปุ่มนำทาง (Anchor Links) และฟอร์มสำหรับสั่งซื้ออาหารที่ใช้งานได้จริง

---

## ✨ คุณสมบัติของเว็บไซต์ (Features)

* **Responsive Header Image:** แสดงภาพโปรโมทร้านอาหารไทยที่จัดกึ่งกลางอย่างสวยงาม
* **Navigation Bar:** ปุ่มเมนูลัดด้านบน (`Menu` และ `Order Form`) ที่กดแล้วจะเลื่อนหน้าจอไปยังส่วนนั้นๆ ทันทีด้วย HTML Anchor Links
* **Menu Section:** ส่วนแสดงรายการอาหาร (ในโค้ดตัวอย่างคือเมนู ผัดไทย/Pad Thai) พร้อมระบุราคาและปุ่มทางลัดเพื่อกดสั่งซื้อทันที
* **Interactive Order Form:** ฟอร์มสั่งซื้ออาหารที่รองรับการกรอกข้อมูลที่จำเป็น เช่น ชื่อผู้สั่ง, เมนู, จำนวน (จำกัด 0-10 จาน), ที่อยู่จัดส่ง และกล่องติ๊กเลือกยืนยันการชำระเงิน

---

## 🛠️ โครงสร้างเทคโนโลยี (Tech Stack & Structure)

โปรเจกต์นี้เขียนขึ้นด้วยภาษาพื้นฐานของเว็บพัฒนา เพื่อเน้นการทำความเข้าใจโครงสร้างที่ถูกต้อง:

* **HTML5:** ใช้ในการวางโครงสร้าง Semantic Tags เช่น `<header>`, `<main>`, `<section>`, `<article>`, และ `<form>`
* **CSS (Internal Style):** ตกแต่งหน้าตาของ Navbar, ลิงก์ตอนที่เมาส์ชี้ผ่าน (`:hover` ให้เปลี่ยนเป็นสีส้ม), และการจัดตำแหน่งรูปภาพ

---

## 📂 รายละเอียดโครงสร้างโค้ด (Code Overview)

เว็บไซต์นี้ประกอบด้วยส่วนสำคัญ 3 ส่วนหลักในโครงสร้าง HTML:

### 1. Head & Navigation (`<head>`)
* บรรจุ Metadata และ `<style>` สำหรับควบคุมหน้าตาของระบบนำทาง
* มีคลาส `.navbar` และ `.order` สำหรับจัดการสีและการแสดงผลของลิงก์

### 2. Menu Section (`<section id="#heading-1">`)
* ใช้แท็ก `<article>` ในการแบ่งกลุ่มข้อมูลของแต่ละเมนูอาหาร
* เชื่อมโยงปุ่ม "Order Now" ตรงไปยังฟอร์มสั่งซื้อด้านล่าง

### 3. Order Form (`<form id="#heading-2">`)
* ใช้ Attributes ของ HTML5 input เช่น `required` เพื่อป้องกันการส่งฟอร์มว่าง
* ใช้ `type="number"` พร้อมกำหนด `min="0"` และ `max="10"` เพื่อจำกัดจำนวนการสั่งซื้อ

## 🚀 วิธีการนำไปใช้งาน (How to Run)

1. ดาวน์โหลดหรือคัดลอกซอร์สโค้ดไฟล์ `index.html` ไปไว้ในเครื่องคอมพิวเตอร์ของคุณ
2. ดับเบิลคลิกที่ไฟล์ `index.html` เพื่อเปิดใช้งานผ่านเว็บเบราว์เซอร์ใดก็ได้ (เช่น Google Chrome, Safari, Firefox)
3. ทดลองคลิกที่ Navbar เพื่อดูการเลื่อนหน้าจอ และทดลองกรอกข้อมูลในฟอร์มสั่งซื้อ

## 📝 บันทึกการแก้ไขเพิ่มเติม (Future Improvements / Bug Fixes)
* **Fix Anchor Links:** ปรับปรุงค่าใน Attribute `id` ของฟอร์มและเซกชัน (ใน HTML ดั้งเดิมมีการใส่เครื่องหมาย `#` ไว้ในไอดี เช่น `id="#heading-1"` ซึ่งที่ถูกต้องควรเป็น `id="heading-1"`) เพื่อให้ระบบ Scroll-to-view ทำงานได้สมบูรณ์แบบ 100%
* **Add CSS File:** ย้ายสไตล์จาก Internal CSS ไปเป็น External CSS (`style.css`) เพื่อความสะอาดของโค้ด
